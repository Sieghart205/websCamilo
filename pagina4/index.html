HOLA MUNDO


a la hora de iniciarse en el mundo de la programacion, normalmente se inicia con lo que se denomina un "hola mundo", que consiste en lograr que en pantalla se imprima la palabra "Hola Mundo", esto solo tiene como fin iniciar a estudiar la sintaxys, por ejemplo: en javascript, seria console.log("hola mundo"); o alert("hola mundo");, en lenguajes como Python, seria print("hola mundo"), en php seria echo "hola mundo", y un extenso etc. normalmente con esto no solo se aprende como mostrar cosas en pantalla, sino que tambien tiene como fin el uso de algunas variables, aunque si se entiende la logica, despues solo es ver como aplicarlo a otro lenguaje, por eso normalmente para iniciarse en programacion se aprende javascript, o Python, que son los dos lenguajes mas conocidos



¿CON QUE LENGUAJE APRENDER?


normalmente se empieza a aprender programacion con Python, aparte de que es un lenguaje muy poderoso, su sintaxis es muy senzilla, no tienes que preocuparte por poner el ; corchetes, o parentecis, Python fue pensado para ser intuitivo y facil, aunque normalmente se usa para analizis de datos, tiene capacidad para crear interfaces graficas, aplicaciones por consola, y hasta puede ser usado para paginas webs, este lenguaje normalmente es usado para desarrollar la logica de programacion, y despues se aprende alguno con enfoque en el area deseada
